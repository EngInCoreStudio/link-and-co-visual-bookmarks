üì¶Creating Containers

What are Containers?

Containers are customizable boxes that hold your bookmarks. Think of them as folders or categories for your links.

How to Create a Container:

  1. Click the + button in the toolbar (requires PRO)
  2. A new container appears on your workspace
  3. Click the container name to rename it
  4. Drag the edges to resize it
  5. Drag the header to move it around

Container Features:

  • Rename: Click on the container title to edit its name
  • Resize: Drag from the bottom-right corner
  • Move: Drag the container header
  • Copy: Click the copy icon (üìã) on the container
  • Delete: Click the trash icon (üóëÔ∏è) on the container
  • Change Color: Use the dropdown menu to select container background colors

üîóAdding Links

Method 1: Drag and Drop

  1. Copy a URL from your browser address bar or any webpage
  2. Drag the link and drop it into a container
  3. The favicon and name are automatically detected

Method 2: Paste URL

  1. Copy a URL to your clipboard
  2. Click inside a container
  3. The extension will detect the URL and create a link

Method 3: Drag from Address Bar

  1. Open the website you want to bookmark
  2. Drag the favicon or URL from the address bar
  3. Drop it into any container or the drop zone
‚ú® Feature: Favicons are automatically fetched from websites. If an icon doesn't appear, try refreshing the page.

üîßTroubleshooting

Links Not Saving?

  • Make sure you're dropping links inside containers, not on the background
  • Check that you have a valid URL in your clipboard
  • Try refreshing the extension

Favicons Not Loading?

  • Some websites block favicon requests
  • Try refreshing the page
  • You can manually set a custom icon URL

Lost Your Data?

  • Check if you have a recent backup file
  • Use "Import data" to restore from backup
  • Contact support if you need assistance

Trial Period Issues?

  • Click "Refresh status" to update your license state
  • Make sure your system clock is correct
  • Contact support if problems persist
